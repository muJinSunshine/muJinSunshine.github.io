requests的基本使用

1.最基本的发送get请求：
import requests
response = requests.get(url)
2.添加headers和查询参数
headers是字典类型的，添加的参数也是字典类型的params
kw = {“wd”: “中国”}
headers = {“User-Agent”: “Mozilla/5.0….”}
response = requests.get(url, params=kw,headers=headers)
查看响应的内容：
print(response.content) 返回的是字节流数据
print(response.text) Unicode数据
查看url完整的地址：
response.url
查看编码：
response.encoding
响应状态码：
response.status_code
